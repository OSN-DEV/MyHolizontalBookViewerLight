# 概要
自分向けの書籍ビューア。最終的には圧縮ファイルを本棚的な感じで書籍管理したいのですが一気に作成できないので、まずは超簡単なビューアを作成。


# 使い方
## ファイルの読込
meta.json のドラッグ・ドロップで書籍を読込

## カレントページの更新
Ctrl + R で現在表示しているページの再読み込み

## メタデータの更新
Ctrl + Shift + R でメタデータの再読み込み

## ページ送り
Ctrl + S・D でページ移動

## 目次表示
Ctrl + O で目次一覧を表示


# meta.jsonの定義
設定内容は meta_schema.md を参照。
